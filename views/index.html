Sample node.js application hosted on Google App Platform (GCP)

Instructions ---

Follow PHP app
Download sample Google Apps Samples
Create a directory (anywhere) on your commputer
as node.js applcation are from the express serverr
Create a default app.js file and place at the same location as package.json
run application usng npm start
run application using npm start
Create an app.yaml file (indentation matters)

    runtime: nodejs20
Search for sample app.yaml files
Place app.yaml into the root directory of your project (same level as index.php)

gcloud commands

gcloud auth login (needed to authorize via browser)
gcloud config set project (select your project)
gcloud app deploy (will upload and instantiate your project)
the first time will ask for region to deploy (ie us-west-1c)
from browser go to your project url of the form <project_id>.uw.r.appspsot.com
This application is at http://phpcabrillo.uw.r.appspot.com

Notes

Once GCP SDK is installed just need to update to get the latest version (gcloud components update)
All development can be done on your local computer
Use preferred text editor on MacOS/Windows/Linux
Use github to save code
After code changes use gcloud app update to upload to App Engine
Sometimes will need to re-authenticate using gcloud command
